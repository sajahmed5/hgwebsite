# Accreditation & affiliation logos

Drop the official logo image files into **this folder** (`public/logos/`) using the
exact filenames below. The site shows each logo automatically; until a file is
added, a brand-styled fallback badge is shown instead, so nothing looks broken.

PNG (transparent background preferred) or SVG. Roughly square or landscape is fine —
each logo is scaled to fit. Aim for ~300px tall or larger.

| Filename                    | Logo                                   |
| --------------------------- | -------------------------------------- |
| `cqc.png`                   | Care Quality Commission                |
| `living-wage.png`           | Living Wage Foundation                 |
| `homecare-association.png`  | Homecare Association                    |
| `antz.png`                  | Antz                                   |
| `iso-45001.png`             | ISO 45001 (Health & Safety)            |
| `iso-9001.png`  (optional)  | ISO 9001 (Quality) — fallback badge for now   |
| `iso-14001.png` (optional)  | ISO 14001 (Environment) — fallback badge for now |
| `iso-27001.png` (optional)  | ISO 27001 — shown as "coming soon"     |

After adding files, commit & push (or upload them via GitHub) and the site redeploys
with the real logos.
